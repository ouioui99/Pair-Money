# Pair-Money

## 現状課題

1. 毎月の楽天ペイ以外でそれぞれが支払った金額の集計が面倒
2. 家賃、水道光熱費、楽天ペイ、三井カードそれぞれの合計金額の集計が面倒
3. いくら何に使ったのかが見えにくい

## 機能

1. レシート読み込み機能 (1)
2. それぞれの使用金額入力機能 (1)
3. 使用総金額集計機能 (2)
4. 固定費登録機能 (ex)
5. 使用額の種類分け機能 (3)
6. 使用額の種類登録機能 (3)
7. ログイン/ユーザ登録機能 (1)
8. 一人が支払う金額表示機能 (2)

## 画面コンポーネント

2. 固定費編集画面
